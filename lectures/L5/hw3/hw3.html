    • Implement the following sort algorithms:

      • bubbleSort:public static < T extends java.lang.Comparable< ? super T>>void bubbleSort(T[] a)
      • selectionSort :public static < T extends java.lang.Comparable< ? super T>>void selectionSort(T[] a)
      • insertionSort : public static < T extends java.lang.Comparable< ? super T>>void selectionSort(T[] a)
      • heapSort: public static < T extends java.lang.Comparable< ? super T>> void heapSort(T[] a)
      • mergeSort

        • standard recursive mergesort : public static < T extends java.lang.Comparable< ? super T>> void mergeSort(T[] a,T[] tmpArray)
        • recursive mergesort switching to insertionSort when size is less than or eqaul to a cutoff size :
          public static < T extends java.lang.Comparable< ? super T>> void mergeSort(T[] a,T[] tmpArray, int cutoff)

      • quicksort

        • standard recursive quicksort, using the leftmost element as the pivot element
          public static < T extends java.lang.Comparable< ? super T>> void quicksort(T[] array)
        • recursive quicksort, using the leftmost element as the pivot element , switch to insertionsort when size is less than or equal to a cutoff size.
          public static < T extends java.lang.Comparable< ? super T>> void quicksort2(T[] a)
        • recursive quicksort, using the median of three as the pivot element , switch to insertionsort when size is less than or equal to a cutoff size.
          public static < T extends java.lang.Comparable< ? super T>> void quicksort3(T[] a)
        • non-recursive quicksort,using the median of three as the pivot element, switch to insertionsort when size is less than or equal to a cutoff size.
          public static < T extends java.lang.Comparable< ? super T>> void quicksort4(T[] a)
