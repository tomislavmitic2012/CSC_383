Recursion
  • A method (function) calling itself is called a recursive method.

  • Examples

    • Factiorial :
      • recurisve version 

             public static long factorial(int n) {

                if ( n == 0 ) return 1 ;
                return n*factorial(n-1) ;
    }

      • iterative version 

              public static long fact(int n) {

          long result = 1 ;

          for ( int i = 1 ; i <= n ; i++ )
                result = result * i;
          return result ;
      }

  • Classification of recursions

    • Linear recursion : At most one recursive call each time it is invoked.
      • Example : Summing the elements of an integer array

        • recursive 
                 public static int sum(int [] A, int n ) {

             if (n == 1 ) return A[0] ;
             return A[n-1] + sum(A,n-1);
        }

        • iterative 

                  public static int sum(int [] A, int n ) {

             int result = 0 ;

             for (int i = 0 ; i < n ; i++ )
                         result += A[i] ;
              return result ;
       }
      • Example : reversing an array

        • Recursive 

          
        public static void reverse(int [] A, int i , int j ) {
                int temp ;

                  if ( i <  j )  {
                            temp = A[i] ;
                               A[i] = A[j] ;
                               A[j] = temp ;
                   }
                   reverse(A, i+1, j-1) ;
        }

        public static void reverse(int [] A) {
                reverse(A, 0,A.length-1) ;
        }

        • iterative 

               public static void reverse(int [] A , int i , int j) {

                int temp ;

                while ( i < j ) {
                     temp = A[i] ; A[i] = A[j] ; A[j] = temp ;

                     i++ ; j-- ;
                 }
      }

      • Tail Recursion : An algorithm is taik recursion if (a) it is a linear recursion and (b) it makes a recursive call as its very last operation

        • Examples:

          • Euclidean algorithm for GCD ; 
            /**
  * Euclid's greatest common divisor algorithm.
  * Find the largest number that evenly divides into both n and d.
  * require: n >= 0, d >= 0.
  * fastest if n >= d.
  */
          public static int gcd(int n, int d){

               if ( d == 0 ) return n;
      
              int r = n % d;

             return( r == 0 ) ? d : gcd( d , r );
       }

      • Multiple Recursion: A method may make multiple ( more than two) recursive calls.
