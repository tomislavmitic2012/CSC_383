Prim’s Algorithm

 

 

              MST_PRIM (G, w, v)

1.      Q ← V[G]

2.      for each u in Q do

3.          key [u] ←  ∞

4.      key [r] ←  0

5.      π[r] ← NIl

6.      while queue is not empty do

7.          u ← EXTRACT_MIN (Q)

8.          for each v in Adj[u] do

9.              if v is in Q and w(u, v) < key [v]

10.              then π[v] ← w(u, v)

11.                  key [v] ← w(u, v)
